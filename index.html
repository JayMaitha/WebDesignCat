My Portfolio

Welcome to my portfolio! I’m Jay Maitha, a passionate website developer and dedicated ICT student at Mount Kenya University. As I delve into web development, I focus on creating user-friendly, responsive websites that provide impactful online experiences. Here, you’ll find a selection of projects I’ve worked on, showcasing my growing expertise in front-end and back-end development.

Project 1. Title: Personal Blog Website

Description: This project was a custom-built blog website designed to help users easily share personal stories, travel experiences, and insights. The website has a simple and intuitive layout, with custom styling to make navigation seamless. Users can easily create posts, manage content, and engage with readers through the comments section, creating a complete online blog experience.

Project 2. Title: E-Commerce Storefront

Description: I developed an e-commerce storefront focused on simplicity and ease of use. This website includes product pages, a shopping cart, and a secure checkout process. The design emphasizes mobile responsiveness, making it accessible across various devices. This project strengthened my skills in web design, payment integration, and user experience optimization.

Project 3. Title: Portfolio Website for a Photographer

Description: I created a personal portfolio site for a photographer, providing an elegant gallery layout that highlights their work. The website has a minimalist design to keep the focus on images, along with interactive features for image previews and a contact form. This project emphasized both aesthetic appeal and functionality, enhancing the photographer’s online presence.
